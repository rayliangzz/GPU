## Overview
---

While GPUs are commonly known as graphics processors, many applications benefit from the highly parallelized architecture inherent in their design. This project is a basic implementation of a General Purpose GPU (GPGPU), a type of processor that aims to accelerate matrix math calculations. 

### Motivation
---

The goal of the Boring Club GPU project is to introduce digital hardware design concepts with a project that is specifically relevant to the current world market.
